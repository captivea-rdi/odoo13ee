Connect your Office 365 Calendar with Odoo

By installing this module you provide your Odoo users a way to sync their Odoo and Outlook calendar. After setup, the synchronisation will happen automatically, without the need of any user interaction, providing the Odoo users an effortless experience.

Features

Automatic two-way synchronisation of your Odoo and Outlook calendar

Automatic synchronisation of changes made in Outlook and Odoo, without the need of any kind of user input.

Easy user login and setup

User setup has been made easy and secure with the use of Oauth2. After signing in with the Office 365 account, an Outlook calendar can be chosen to sync your personal Odoo calendar with.

By selecting the "Only sync calendar items with category" option, Odoo will only look at the items that have the provided category in Outlook.

Changes made in either Odoo or Outlook will be synced in the background, and do not need any further user input.

Manual sync option

In case of changes that need to be synced immediately between your Odoo and Outlook account, a handy "Sync Calendar" button has been provided for instant calendar updating.

Support for recurring items in Outlook and Odoo

Events or appointments that are recurring in Odoo or Outlook are synced like any other item.

Easily sync new items in Outlook to Odoo

Items such as events and appointments created in the Outlook calendar will be automatically synced with Odoo. If the user has selected the option to only sync items with the provided category, manually adding the category to the item in Outlook will initiate the synchronisation.

Requirements

The following requirements need to be met before using this module:

  • Your Odoo installation is accessible via HTTPS and a public DNS name
  • Your Company is making use of Microsoft Office 365 accounts
  • The base framework module office365_framework has been setup correctly
