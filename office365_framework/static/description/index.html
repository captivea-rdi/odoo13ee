Connect your Office 365 Account with Odoo

< class="oe_mb32"> This module does not provide any functionality on it's own. It is meant as the base framework for other modules that want a two-way sync between Odoo and Office 365.

Checkout the list below for modules that depend from this framework.

Implementations

Office 365 Contact Synchronisation

Provides two-way synchronisation between Odoo and Outlook contacts.

Office 365 Calendar Synchronisation

Provides two-way synchronisation between Odoo and Outlook calendar items.

Features

Automatic two-way synchronisation of your Odoo and Outlook items

Automatic synchronisation of changes made in Outlook and Odoo, without the need for user input.

Easy user login and setup

User setup has been made easy and secure with the use of Oauth2. After signing in with the Office 365 account, an Outlook calendar can be chosen to sync your personal Odoo calendar with.

Requirements

The following requirements need to be met before using this module:

  • Your Odoo installation is accessible via HTTPS and a public DNS name
  • Your Company is making use of Microsoft Office 365 accounts
